Battlebot - Privacy Policy

Last Updated: November 2025

1. Overview

This Privacy Policy explains what information Battlebot ("the Bot") collects, how that information is used, and the rights users have regarding their data. By using the Bot, you agree to this Privacy Policy.

2. Information the Bot Collects

Battlebot collects only the minimum information required to operate its features. This includes:

  • Discord User IDs (unique numeric identifiers)
  • Discord Guild (Server) IDs
  • Troop information voluntarily submitted by users:
    • Troop type (Fighter, Shooter, Rider)
    • Troop level
    • Troop Combat Power (CP)
    • March size
  • Rally configuration data (slots and rally size)
  • Event participation data (RSVP choices)

The Bot does NOT collect:

  • Messages or message content
  • Private messages (DMs)
  • Usernames or nicknames
  • Email addresses, IP addresses, or any identifying info
  • Server logs or activity history

3. How Information Is Used

Stored data is used exclusively for game-planning features, including:

  • Generating combat rally plans
  • Coordinating event participation
  • Generating base layouts for strategic placement
  • Storing user-selected troop data for future calculations

Information is never shared with third parties, and is never sold.

4. Data Storage

  • All data is stored in a secure private database controlled by the developer.
  • Data is isolated per Discord server using Discord’s guild ID system.
  • No cross-server linking or global profiling occurs.

5. Data Retention

Battlebot retains user data only as long as it is needed for active server use. Inactive or unused data may be removed during maintenance updates.

6. Data Removal Requests

Users may request removal of their stored data at any time by:

  • Contacting the developer directly via Discord
  • Asking their server administrator to remove their data

Upon request, all data tied to the user’s Discord ID will be permanently deleted.

7. Children’s Privacy

Battlebot does not knowingly collect data from children under 13, in compliance with Discord’s Terms of Service. If such data is discovered, it will be removed immediately.

8. Security

Reasonable technical measures are implemented to protect stored data from unauthorized access or misuse. However, no system is 100% secure, and use of the Bot is at the user’s own risk.

9. Changes to This Privacy Policy

This Privacy Policy may be updated periodically. Continued use of the Bot after updates indicates acceptance of the revised policy.

10. Contact

For data removal, concerns, or questions, please contact the Bot developer via Discord.
