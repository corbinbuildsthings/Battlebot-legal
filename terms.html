2. Description of Service

Battlebot is a Discord bot designed to assist with game coordination, including managing troop information, planning combat rallies, generating base layouts, creating event polls, and other game-related calculations.

3. Data the Bot Collects

The Bot stores only the minimum data required for its features, including:

  • Discord User IDs

All stored data is used solely to provide in-server features such as combat planning, base planning, and event coordination. Data is not shared with third parties and is stored server-by-server (no cross-server linking).
